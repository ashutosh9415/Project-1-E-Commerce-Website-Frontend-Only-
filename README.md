# E-Commerce Website (Frontend Only)

## Project Description

Built a responsive and modern E-Commerce website inspired by popular platforms like Amazon and Flipkart using frontend technologies. The application allows users to browse products, search and filter items, view detailed product information, manage shopping carts, and maintain wishlists with an interactive and user-friendly interface.

## Features Implemented

- Responsive product listing page with grid layout
- Product detail page with complete product information
- Add to Cart and Remove from Cart functionality
- Dynamic cart total calculation
- Product search functionality
- Category, price, and rating filters
- Wishlist feature
- Product image carousel/slider
- Skeleton loading screens for better user experience
- Dark and Light mode toggle
- Fully responsive design for mobile, tablet, and desktop devices

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- React.js
- React Router DOM
- Context API / Redux (for state management)
- Bootstrap / Tailwind CSS
- Local Storage
